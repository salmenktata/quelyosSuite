# Guide de Test - Module E-commerce Quelyos

## ✅ Tests Backend Odoo

### 1. Accès Interface Backoffice

**URL:** http://localhost:8069

**Identifiants par défaut:**
- Email: `admin`
- Mot de passe: `admin`

### 2. Vérifications Interface Odoo

Une fois connecté, vérifier que les menus suivants sont présents:

#### Menu E-commerce Principal
- **E-commerce** (menu principal)
  - **Catalogue** → **Produits**
  - **Commandes** → **Toutes les Commandes**  
  - **Commandes** → **Paniers Abandonnés**
  - **Marketing** → **Avis Produits**
  - **Marketing** → **Coupons**
  - **Analytics** → **Dashboard**

✅ Menu E-commerce visible

✅ Création coupon OK
