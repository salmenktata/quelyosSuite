# Logos Quelyos

⚠️ **IMPORTANT**: Placez vos logos Quelyos dans ce dossier avant d'installer le module.

## Fichiers requis

### Logos principaux (HAUTE PRIORITÉ)

1. **quelyos_logo.png**
   - Dimensions: 1000x250px
   - Format: PNG avec fond transparent
   - Utilisation: Site web, emails, POS
   - Logo en couleur (bleu #1e40af)

2. **quelyos_logo_white.png**
   - Dimensions: 1000x250px
   - Format: PNG avec fond transparent
   - Utilisation: Navbar backend, page de connexion
   - Logo en blanc (#FFFFFF)

3. **quelyos_logo_small.png**
   - Dimensions: 180x46px
   - Format: PNG avec fond transparent
   - Utilisation: Navbar (version compacte), tickets POS
   - Logo blanc pour navbar, logo couleur pour tickets

4. **quelyos_logo.svg** (OPTIONNEL)
   - Format: SVG vectoriel
   - Utilisation: Version scalable pour tous usages
   - Permet un meilleur rendu à toutes les tailles

## Spécifications techniques

- **Formats acceptés**: PNG (recommandé), SVG
- **Fond**: Transparent (PNG)
- **Couleurs**:
  - Logo couleur: Bleu #1e40af (primaire)
  - Logo blanc: #FFFFFF
- **Qualité**: Haute résolution (300 DPI recommandé pour PNG)
- **Poids**: < 500 KB par fichier

## Où sont utilisés les logos ?

| Fichier | Utilisations |
|---------|--------------|
| `quelyos_logo.png` | Site web header, emails, portail client, POS |
| `quelyos_logo_white.png` | Navbar backend (fond bleu), page de connexion |
| `quelyos_logo_small.png` | Navbar compacte, tickets de caisse POS |
| `quelyos_logo.svg` | Version vectorielle pour tous usages |

## Création des logos

Si vous n'avez pas encore les logos, vous pouvez:

1. **Designer professionnel**: Engager un designer pour créer votre identité visuelle
2. **Outils en ligne**: Utiliser Canva, Figma, Adobe Express
3. **Templates**: Partir d'un template et le personnaliser
4. **IA**: Utiliser des outils comme Midjourney, DALL-E pour générer

### Checklist de design

- [ ] Logo lisible à petite taille (navbar)
- [ ] Bon contraste sur fond blanc ET fond bleu
- [ ] Design professionnel et moderne
- [ ] Cohérent avec l'identité de marque Quelyos
- [ ] Déclinaison blanche pour fonds sombres
- [ ] Version carrée/compacte pour espaces réduits

## Test des logos

Après avoir ajouté les logos:

1. Redémarrer Odoo
2. Vérifier la navbar backend (logo blanc visible)
3. Vérifier la page de connexion (logo blanc + slogan)
4. Vérifier le site web (logo couleur dans header)
5. Vérifier les emails (logo couleur dans header email)
